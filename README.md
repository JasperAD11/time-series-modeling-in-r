# Time Series Econometrics Assignments

This repository contains two R-based assignments completed for a course in time series econometrics, with a focus on time series structure, residual diagnostics, seasonality, and forecasting.

## Overview

The assignments explore core time series methods using two datasets with different temporal structures:

- **Data 1:** a monthly time series running from 2010 to 2023, used to study trend behavior and forecasting over time
- **Data 2:** a daily time series running from 2019 to 2023, used to examine trend, weekly and monthly seasonality, special events, and forecast performance

The work is implemented in R using reproducible `.Rmd` files and exported reports in PDF format.

## Topics covered

Across both assignments, the repository covers:

- time plots and pattern identification
- trend assessment
- seasonality analysis
- autocorrelation and residual diagnostics
- benchmark forecasting models
- train/test forecast evaluation in R

Assignment 1 focuses on identifying structure in two time series and comparing simple benchmark forecasting models. 

Assignment 2 extends the analysis by testing trend, monthly and weekday seasonal effects, and selected event dummies in a forecasting framework.
